Variables

Datatype: have different data types like string, number, boolean, array and object. You can learn a lot about a variable based on how it looks in the console. Arrays are stored in brackets, Objects are stored in curly braces.

Host objects: document, elements, events

Name: all have identifiers; first character starts with letter a-z, _ or $

Use camelCase, including using camelCase for files e.g. movieSearch.js
